### Generate

```
rails g bower_rails:initialize
      create  bower.json
```
results in a bower.json file containing:

```json
{
  "lib": {
    "name": "bower-rails generated lib assets",
    "dependencies": {
    }
  },
  "vendor": {
    "name": "bower-rails generated vendor assets",
    "dependencies": {
    }
  }
}
```
